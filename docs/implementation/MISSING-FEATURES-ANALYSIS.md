## 7. 우선순위 정리

### 🔴 P0 - 핵심 게임플레이 (즉시 필요)

1. **보충 시스템 완전 구현** - 시간 공식, 락 상태, 취소 불가
2. **도주 시스템** - 빈 보트 탑승, 조건 체크
3. **유닛 선택 시 자동 슬로우 모션**
4. **지휘관 무적 조건** - 병사 1명+ 생존 시
5. **명령 버튼 3개** - 도주/회복/이동

### 🟠 P1 - 중요 피드백 (다음 우선)

6. **녹색 원형 선택 아이콘**
7. **웨이브 방향 화살표**
8. **FINAL WAVE 텍스트**
9. **피 스플래터 + 시체 지속**
10. **침투선 물결 효과**

### 🟡 P2 - 폴리시 (나중)

11. 지휘관 깃발 비주얼
12. 날씨 효과 (비, 황혼)
13. 건물 화재 애니메이션
14. 숫자 키 1-4 단축키
15. 카메라 회전 버튼

### 🟢 P3 - 선택적 (여유 있을 때)

16. 체크포인트 시스템
17. 골드 수동 분배 UI
18. 섬 이름 필기체
19. 다른 헬멧 = 업그레이드 신호
