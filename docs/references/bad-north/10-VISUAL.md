## 유닛 비주얼 ✅

### 아군 (파란색 계열)

| 유닛 | 시각적 특징 |
|------|------------|
| 창병 (Pikes) | 긴 창 (유닛 높이의 **~1.5배**), 파란색 망토 |
| 보병 (Infantry) | 방패 + 검, 파란색 망토 |
| 궁수 (Archers) | 활, 파란색 망토 |
| 지휘관 | **분홍색 깃발** 부착, 문양 표시 |

### 적 (검은색/갈색 계열)

| 유닛 | 시각적 특징 |
|------|------------|
| 일반 바이킹 | 검은 옷, 검만 소지 |
| 방패병 | 노란색/갈색 원형 방패 |
| 궁수 | 활 소지, 검은 옷 |

## 전투 이펙트 ✅

### 화살

| 요소 | 관측 |
|------|------|
| 시각화 | **밝은 직선** (살구색/노란색) |
| 발사 각도 | 약 **30-45도** 포물선 |
| 착탄 | 적 위치에 즉시 히트 |
| 다중 화살 | 동시에 여러 발 표시 |

### 피/사망

| 요소 | 관측 |
|------|------|
| 피 색상 | **선명한 빨간색** (#C41E3A) |
| 피 형태 | 지면에 **스플래터** 패턴 |
| 시체 | 땅에 쓰러진 형태로 유지 |
| 시체 색상 | 적은 검은색, 피가 묻음 |
| 피 지속 | **전투 끝까지** 지면에 남음 |

## 환경 비주얼 ✅

### 지형 요소

| 요소 | 관측 |
|------|------|
| 타일 | 육각형 느낌의 부드러운 경계 |
| 절벽 | **흰색 수직 면** |
| 경사면 | 계단식 지형 |
| 나무 | 짙은 녹색/검은색 **구체 형태** |
| 집 | 흰색 벽, 회색/갈색 지붕 |
| 덤불 | 연한 녹색, 타일 경계에 배치 |

### 집 크기

| 크기 | 형태 | 골드 |
|------|-----|------|
| 1×1 | 작은 단독 건물 | 1 |
| 2×1 | 긴 직사각형 건물 | 2 |
| 2×2 | L자형 또는 큰 건물 | 3 |

### 날씨 효과

| 날씨 | 시각 표현 |
|------|----------|
| **비** | 물방울 + 세로 빗줄기 + 물결 |
| **맑음** | 기본 상태, 안개 효과 |
| **황혼** | 붉은/보라 톤 |

## 카메라 & 시점

### 기본 시점
- **아이소메트릭 뷰**: 고정된 위에서 내려다보는 시점
- **디오라마 스타일**: 섬 전체를 한눈에 조망

### 카메라 기능
- **회전**: 360도 회전 가능 (드래그 또는 버튼)
- **줌**: 마우스 휠로 줌 인/아웃
- **자동 프레이밍**: 섬 전체가 화면에 맞춰짐

## The Fading Raven 적용 포인트

### 유지해야 할 비주얼 요소
- 깃발 시스템 (지휘관 식별)
- 녹색 원형 선택 표시
- 타일 핀 마커 (이동 가능 위치)
- 피 스플래터 지속 효과
- 미니멀 HUD

### UI 레이아웃 참고
- 월드맵: 노드 기반, 좌→우 진행
- 전투: 디오라마 뷰, 회전 가능
- 업그레이드: 카드형 클래스 선택
- 승리: 골드 분배 UI

### 수치 참고
- 분대 크기: **8-9명** (기본)
- 명령 버튼: **3개** (도주, 회복, 이동)
- 건물 수 (섬당): **3-5채** 평균
