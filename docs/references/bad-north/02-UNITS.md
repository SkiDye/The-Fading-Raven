## 유닛 클래스 (4종)

### Militia (민병대)
- 모든 지휘관의 시작 상태
- 특수 능력 없음
- **6골드**로 클래스 업그레이드

### Infantry (보병) ✅
> "A versatile melee unit carrying shields to block blows and projectiles."

| 속성 | 값 |
|------|-----|
| 역할 | 올라운더, 대궁수 |
| 무기 | 검 + 방패 |
| 스킬 | **Plunge** (급강하) |

**장점**
- 방패: 화살/투척 ~90% 방어 🔵
- 이동 중 공격 가능
- Plunge로 고지대 급습

**단점**
- ⚠️ **근접전 중 방패 무효** - 교전 시 화살 방어 불가
- 브루트에게 약함

### Pikes (창병) ✅
> "Great at holding one position, but cannot fight while moving."

| 속성 | 값 |
|------|-----|
| 역할 | 방어 특화, 대브루트 |
| 무기 | 장창 |
| 스킬 | **Pike Charge** (돌격) |

**장점**
- 긴 리치, 병목 최강
- 브루트 유일한 카운터
- 경사면에서 효과 증가

**단점**
- ⚠️ **이동 중 공격 불가**
- ⚠️ **근접 시 창 들어올림** → 무력화
- 화살에 취약 (방패 없음)

### Archers (궁수) ✅
> "Deals death from afar but struggles with close encounters."

| 속성 | 값 |
|------|-----|
| 역할 | 원거리 딜러 |
| 무기 | 활 |
| 스킬 | **Volley** (일제 사격) |

**장점**
- 상륙 전 적 처치 가능
- 고지대 보너스 (명중률/사거리)
- 적을 물에 밀어넣기 (넉백)

**단점**
- 방패병에게 막힘
- ⚠️ **Elite 전까지 약함** (정확도 낮음)
- 근접에서 무력

## 지휘관 메카닉

### 무적 조건
- 병사 **1명 이상 생존** → 지휘관 무적
- 브루트 공격에도 사망하지 않음

### 분대 전멸 조건
| 상황 | 결과 |
|------|------|
| 전 병사 사망 + 지휘관 고립 | 다음 섬에서 자동 복구 |
| 보충 중 집 파괴 | **영구 손실** 가능 |
| 도주 실패 | **영구 손실** |
