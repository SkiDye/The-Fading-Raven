# Bad North - 캠페인 & 진행 시스템

## 게임 루프

```
[캠페인 맵에서 섬 선택]
        ↓
[섬 방어 전투 수행]
        ↓
[집 방어 성공 → 골드 획득]
        ↓
[유닛 업그레이드/아이템 구매]
        ↓
[다음 섬으로 이동 or 휴식]
        ↓
[바이킹 전선 전진]
        ↓
    (반복)
```

---

## 캠페인 맵 구조

### 기본 구조
- **절차적 생성**: 매 플레이마다 다른 맵
- **진행 방향**: 서→동 (좌→우)
- **다중 경로**: FTL 스타일 노드 선택

### 바이킹 전선
- 점점 동쪽으로 이동
- **뒤로 돌아갈 수 없음**
- 휴식 시 전선 전진

### 시각적 표시 (월드맵)

| 표시 | 의미 |
|------|------|
| 실선 | 현재 바이킹 전선 |
| 점선 | 다음 휴식 시 사라질 섬 |
| 검은 화살 | 적 수량 힌트 (품질 아님) |
| 집 수 | 난이도 + 보상 힌트 |
| **🚩 깃발** | **새 지휘관 영입 가능** (홈 아일랜드) |
| **❓ 물음표** | **아이템 획득 가능** |

### 맵 노드 색상 (관측)

| 색상 | 의미 |
|------|------|
| 빨간색 | 적 점령 |
| 노란색 | 선택됨 |
| 청록색/회색 | 미방문 |

## 섬 유형

| 유형 | 표시 | 특징 |
|------|------|------|
| 일반 전투 섬 | 집 아이콘 | 바이킹 방어 |
| 아이템 섬 | **❓** | 아이템 획득 가능 |
| 지휘관 섬 | **🚩** | 새 지휘관 영입 (홈 아일랜드, 도주 불가) |
| 체크포인트 섬 | - | 진행 저장 (Jotunn Edition) |

### 지휘관 영입 규칙
- 깃발 섬의 지휘관은 **그 섬을 반드시 방어해야** 영입 가능
- 홈 아일랜드에서는 **도주 불가** - 충분한 병력 지원 필수
- 같은 특성의 지휘관은 한 캠페인에 **중복 등장하지 않음**

---

## 턴 시스템

### 기본 규칙
- 턴당 각 지휘관 **1회 전투**만 가능
- 지휘관 수 = 턴당 가능 전투 수
- 휴식 = 턴 종료 + 바이킹 전선 전진

### 전략적 고려
- 지휘관 많으면 턴당 여러 섬 공략 가능
- 골드 수입 극대화
- 전선 밀리기 전에 섬 확보

---

## 골드 경제 시스템

### 골드 획득

**집 방어 보상**

| 집 크기 | 골드 |
|--------|------|
| 1×1 | **1** |
| 2×1 | **2** |
| 2×2 | **3** |

**추가 획득**
- Philosopher's Stone 보너스 (+1~3/섬)

**수입 계산**
```
섬 수입 = Σ(생존 집 골드) + Philosopher's Stone 보너스
```

### Jotunn Edition 골드 시스템

| 변경 전 | 변경 후 |
|--------|--------|
| 개별 지휘관 골드 | **공유 골드 뱅크** |

**장점**
- 전략적 투자 가능
- 특정 지휘관 집중 육성
- 턴당 다중 섬 공략 보상

### 지출 항목

| 항목 | 비용 |
|------|------|
| 클래스 업그레이드 (Militia→Standard) | **6** |
| Veteran | **12** |
| Elite | **20** |
| 스킬 Lv1 | **7** |
| 스킬 Lv2 | **10** |
| 스킬 Lv3 | **14** |
| 아이템 업그레이드 | 5~16 |

### 경제 전략

**원칙**
1. 분산 투자 금지
2. 한 유닛 집중 강화
3. 전투력 > 경제 아이템

**우선순위**
```
1. 핵심 유닛 Veteran
2. 핵심 유닛 Elite
3. 아이템 Lv1
4. 추가 유닛 업그레이드
5. 스킬/아이템 추가 업그레이드
```

## 난이도 시스템

### 난이도 설정

| 난이도 | 해금 조건 | 특징 |
|--------|----------|------|
| Easy | 기본 | 체력/타이밍 완화, 보충 단축, 집 연소 느림 |
| Normal | 기본 | 표준 |
| Hard | 기본 | 적 수 증가 |
| Very Hard | Hard 클리어 | 최고 난이도 |

### 진행 스케일링

| 구간 | 섬 수 | 특징 | 등장 적 |
|------|-------|------|--------|
| 초반 | 1~10 | 기본 적, 2명으로 방어 가능 | Militia, Archers |
| 중반 | 10~20 | 중급 적 등장, 업그레이드 필요 | Shieldmen, Huscarls |
| 후반 | 20+ | 급격한 난이도 상승 | Brutes, Dual Wielders |

### 난이도 상승 신호
- 다른 헬멧 착용 적 등장
- 보트 크기 증가
- 동시 상륙 증가

## 밸런싱 전략

### 초반 (1~10섬)
- 3~4명 **Veteran** 확보
- 클래스 다양화
- 쉬운 섬 우선

### 중반 (10~20섬)
- **Elite** 1~2명 확보
- 아이템 장착
- 지형 활용 숙달

### 후반 (20섬+)
- 풀 **Elite** 필수
- 궁수로 점퍼 대응
- 손실 최소화 운영

---

## 섬 선택 전략

### 고려 요소
1. **집 수**: 많을수록 골드↑, 난이도↑
2. **지형**: 병목 많으면 방어 유리
3. **화살 수**: 적 수량
4. **접근로 수**: 적으면 방어 유리

### 추천 전략
```
쉬운 섬 (집 적음, 병목 많음)
  → 안정적 클리어 + 적은 골드

어려운 섬 (집 많음, 개방적)
  → 위험하지만 높은 보상
```

**원칙**: 쉬운 섬에서 적은 골드 > 어려운 섬에서 전멸

## 영구 사망 (Permadeath)

### 게임 오버 조건
- **모든 지휘관 사망** → 게임 종료

### 지휘관 생존 조건
- 병사 **1명 이상 생존** → 지휘관 무적

### Jotunn Edition 체크포인트
- 특정 섬에서 진행 저장 가능
- 전멸 시 마지막 체크포인트로 복귀
