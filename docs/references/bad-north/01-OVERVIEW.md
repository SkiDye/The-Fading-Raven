## 핵심 컨셉

### 디자인 철학
- **미니멀리스틱 아트**: 단순하고 귀여운 비주얼 vs 잔혹한 전쟁
- **절차적 생성**: 섬, 캠페인 맵 모두 무작위 생성 (WFC 알고리즘)
- **영구 사망**: 지휘관 사망 시 해당 런에서 영구 손실
- **가위바위보 상성**: 유닛 간 명확한 카운터 관계

### 게임 루프
```
[캠페인 맵에서 섬 선택]
        ↓
[섬 방어 전투 수행]
        ↓
[집 방어 성공 → 골드 획득]
        ↓
[유닛 업그레이드/아이템 구매]
        ↓
[다음 섬으로 이동 or 휴식]
        ↓
[바이킹 전선 전진]
        ↓
    (반복)
```

---

## 스토리 배경

왕이 바이킹 침략자들에게 살해당하고, 플레이어는 왕국을 이끌며 백성들의 탈출을 지휘한다. 승리가 아닌 **생존**을 위한 필사적인 방어전.

## The Fading Raven 세계관 매핑

| Bad North | The Fading Raven |
|-----------|------------------|
| 바이킹 왕국 | 성간 연합 |
| 섬 | 우주 정거장 |
| 집 | 시설 모듈 |
| 보트 | 침투선 |
| 물 (즉사) | 우주 공간 |
| 바이킹 | 해적단 + 폭풍 생명체 |
| 지휘관 | 크루 리더 |
