# Bad North PC 조작법 리서치

## 요약
Bad North는 미니멀리스틱 리얼타임 전술 로그라이트 게임으로, PC 버전에서는 마우스와 키보드 또는 마우스만으로 플레이 가능하다. 게임 내 별도의 조작법 설명 화면이 없어 커뮤니티 정보에 의존해야 한다.

---

## 유닛 컨트롤

### 유닛 선택
| 조작 | 방법 |
|------|------|
| 마우스 클릭 | 유닛(분대) 직접 클릭으로 선택 |
| 숫자 키 1-4 | 각 분대 빠른 선택 (1번부터 4번 분대) |

> **참고:** 숫자 키 바인딩은 고정되어 있으며 재설정 불가
> 
> **출처:** Steam Discussion - "A list of the keyboard hotkeys might be useful. At least on the Discord version there is NO screen that lists what keyboard keys do. Also, there's no way to rebind said keys. So for example number keys 1-4 serve as hotkeys for your four squads that you control."
> https://steamcommunity.com/app/688420/discussions/0/1748980761812813810/

### 이동 명령
| 조작 | 방법 |
|------|------|
| 마우스 좌클릭 | 선택한 유닛을 목표 타일로 이동 |
| 드래그 | 유닛 선택 후 목표 위치로 드래그 |

## 카메라 컨트롤

### 카메라 회전
| 조작 | 방법 |
|------|------|
| 회전 버튼 클릭 | 빠르게 90도 회전 (스냅 회전) |
| 회전 버튼 홀드 | 천천히 부드럽게 회전 |
| Q / E 키 | 카메라 좌/우 회전 (추정) |

> **참고:** 카메라 회전에 대한 피드백에서 "buttons of rotating the camera rotate slowly if held and spin to win if pressed" 언급
> 
> **출처:** Steam Discussion - "I do not understand what happened during the development of this game, but I cannot imagine why would anyone go 'yes, lets have the buttons of rotating the camera rotate slowly if held and spin to win if pressed'."
> https://steamcommunity.com/app/688420/discussions/0/1741100729950241210/

### 줌 인/아웃
| 조작 | 방법 |
|------|------|
| 마우스 휠 위로 | 줌 인 |
| 마우스 휠 아래로 | 줌 아웃 |

> **출처:** Steam Discussion - 줌 버그 관련 토론에서 "Try to zoom using the mouse wheel" 언급
> https://steamcommunity.com/app/688420/discussions/0/1741104717703229728/

### 카메라 각도
- **아이소메트릭 뷰:** 고정된 위에서 내려다보는 시점
- **회전 범위:** 360도 (4방향 또는 부드러운 회전)
- **틸트/패닝:** 기본적으로 자동 (섬 전체가 화면에 맞춰짐)

## 스킬 사용

| 조작 | 방법 |
|------|------|
| 유닛 선택 후 스킬 버튼 클릭 | 스킬 활성화 |
| 목표 지점 클릭 | 스킬 발동 위치 지정 |

### 스킬 종류
- **Volley (궁수):** 화살 집중 사격
- **Pike Charge (창병):** 돌격
- **Plunge (보병):** 다이브 공격

## UI 버튼

| 버튼 | 기능 |
|------|------|
| Deploy | 유닛 배치 |
| Skip Turn | 다음 웨이브로 건너뛰기 |
| Flee | 섬 탈출 (포기) |

> **주의:** UI 버튼이 화면의 상당 부분을 차지하여 실수로 클릭할 수 있음
> 
> **출처:** Steam Discussion - "the deploy button and skip turn button are the worst culprits here. Again, I have lost islands to accidentally pressing these buttons cause they are taking up about 10% of my screen."

## 검증이 필요한 항목

다음 항목들은 추가 검증이 필요합니다:

1. **카메라 회전 키보드 단축키:** Q/E 또는 A/D인지 확인 필요
2. **마우스 드래그 카메라 회전:** 우클릭+드래그 또는 중클릭+드래그 가능 여부
3. **키보드 리바인딩:** 완전히 불가능한지 확인
4. **컨트롤러 지원:** PC 버전에서 컨트롤러 사용 시 조작법

## 한계점

이 문서는 다음과 같은 한계가 있습니다:

1. **공식 문서 부재:** Bad North는 게임 내 조작법 안내 화면이 없으며, 공식 키바인딩 문서도 제공되지 않음
2. **커뮤니티 의존:** 대부분의 정보가 Steam 토론, 가이드, Reddit 등 커뮤니티 소스에서 수집됨
3. **버전 차이:** Discord 버전, Steam 버전 등 플랫폼에 따라 약간의 차이가 있을 수 있음
4. **Jotunn Edition 업데이트:** 원본 Bad North와 Jotunn Edition 사이에 변경사항이 있을 수 있음
