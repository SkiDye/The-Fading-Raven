> **작성일**: 2026-02-04
> **분석 대상**: 스크린샷 28장+, 전투 프레임 38장+

## 1. 유닛 관측 데이터

### 1.1 분대 크기 (프레임 카운팅)

| 유닛 타입 | 관측된 수 | 비고 |
|----------|----------|------|
| 창병(Pikes) | **8-9명** | attack-001~030 프레임에서 카운팅 |
| 적 바이킹 | **3-5명** (소형 보트) | 개별 보트당 |
| 아군 2분대 | 각 **8-9명** | 두 깃발 확인 |

### 1.2 유닛 시각적 구분

**아군 (파란색 계열)**
- **창병**: 긴 창 (약 유닛 높이의 1.5배), 파란색 망토
- **보병**: 방패 + 검, 파란색 망토
- **궁수**: 활, 파란색 망토
- **지휘관**: 깃발 들고 있음 (분홍색 깃발에 "JU" 같은 문양)

**적 (검은색/갈색 계열)**
- **일반 바이킹**: 검은 옷, 검만 소지
- **방패병**: 노란색/갈색 원형 방패
- **궁수**: 활 소지, 검은 옷

### 1.3 깃발 시스템

| 요소 | 관측 |
|------|------|
| 아군 깃발 색상 | **분홍색** (핫핑크) |
| 깃발 위치 | 지휘관 등 뒤에 부착 |
| 선택 표시 | 깃발 위 **녹색 원형 아이콘** |
| 비선택 시 | 깃발만 보임 (녹색 표시 없음) |

## 2. 전투 메카닉 관측

### 2.1 화살 표현

| 요소 | 관측 |
|------|------|
| 화살 시각화 | **밝은 직선** (살구색/노란색) |
| 발사 각도 | 약 **30-45도** 포물선 |
| 착탄 | 적 위치에 즉시 히트 |
| 다중 화살 | 동시에 여러 발 표시 |

### 2.2 피/사망 표현

| 요소 | 관측 |
|------|------|
| 피 색상 | **빨간색** (선명함) |
| 피 형태 | 지면에 **스플래터** 패턴 |
| 시체 | 땅에 쓰러진 형태로 유지 |
| 시체 색상 | 적은 검은색, 피가 묻음 |
| 피 지속 | 전투 끝까지 지면에 남음 |

### 2.3 보트 시스템

| 요소 | 관측 |
|------|------|
| 보트 색상 | **갈색/적갈색** (바이킹 롱쉽 스타일) |
| 보트 크기 | 소형(3-5명), 대형(8명+) |
| 접근 표시 | **흰색 물결** 효과 (wake) |
| 정박 위치 | 해안가 타일에 붙음 |
| 빈 보트 | 적이 내린 후 해안에 대기 |

### 2.4 상륙 시퀀스 (프레임 분석)

```
프레임 001: 보트 접근 중, 물결 효과
프레임 010: 보트 해안 도착, 적 하선 시작
프레임 020: 적 상륙 완료, 전투 개시
프레임 030: 전투 진행, 피 스플래터 증가
```

## 3. UI/HUD 관측

### 3.1 월드맵 UI (bad-north-map.jpg)

| 요소 | 관측값 |
|------|--------|
| 섬 노드 색상 (적 점령) | **빨간색** |
| 섬 노드 색상 (선택됨) | **노란색** |
| 섬 노드 색상 (미방문) | **청록색/회색** |
| 경로 표시 | 점선(미래), 실선(가능) |
| 커맨더 수 표시 | "4/4 Commanders Available" |
| 턴 버튼 | "Next Turn" + **Y키** |
| 일시정지 | 우상단 **\|\|** 아이콘 |

**섬 이름 예시**: Stack, Coll, Tower Peak, Westray, Samson, Norby, Dusk, Raasay, Shapinsay, Kalsoy, Iron Craig, Svinoy, Sousay

### 3.2 전투 HUD

| 요소 | 위치 | 관측 |
|------|------|------|
| 커맨더 초상화 | 좌하단 | 원형, 색상 배경 |
| 커맨더 이름 | 초상화 아래 | "Idelisa" |
| 명령 버튼 3개 | 초상화 우측 | 깃발, 십자, 핀 아이콘 |
| Move 버튼 | 활성화 시 | **노란색 원형** 하이라이트 |
| 일시정지 | 우상단 | **\|\|** 아이콘 |

### 3.3 명령 버튼 분석

| 아이콘 | 추정 기능 | 관측 |
|--------|----------|------|
| 깃발 + 배 | **Flee (도주)** | 왼쪽 버튼 |
| 십자가 + 집 | **Replenish (회복)** | 중앙 버튼 |
| 핀 마커 | **Move (이동)** | 오른쪽 버튼, 노란색 활성화 |

### 3.4 유닛 선택 피드백

| 상태 | 시각 표현 |
|------|----------|
| 선택됨 | 유닛 위 **녹색 원형 아이콘** |
| 이동 가능 | 타일에 **흰색 핀 마커** |
| 비선택 | 기본 상태 (표시 없음) |

## 10. The Fading Raven 적용 포인트

### 10.1 유지해야 할 비주얼 요소
- 깃발 시스템 (지휘관 식별)
- 녹색 원형 선택 표시
- 타일 핀 마커 (이동 가능 위치)
- 피 스플래터 지속 효과
- 미니멀 HUD (좌하단 커맨더, 우상단 일시정지)

### 10.2 수치 참고
- 분대 크기: **8-9명** (기본)

- 클래스 업그레이드 비용: **6코인**
- 오디오 볼륨 범위: **0-10**

- 월드맵: 노드 기반, 좌→우 진행
