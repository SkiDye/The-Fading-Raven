# Bad North - 지휘관 특성 (Traits)

## 개요

- Jotunn Edition에서 추가
- 총 **13종**의 패시브 버프
- 지휘관 영입 시 **무작위 부여**
- 클래스/아이템과 시너지 고려 필요
- 같은 특성의 지휘관은 한 캠페인에 **중복 등장하지 않음**

---

## 전투 특성 (6종)

### Sharp Weapons (날카로운 무기) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 데미지 증가, 넉백/스턴 감소 |
| 추천 | **창병** |
| 이유 | 빠른 처치 = 방어선 유지 |

---

### Heavy Weapons (무거운 무기) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 넉백/스턴 증가 |
| 추천 | **궁수** |
| 이유 | 방패병도 물에 밀어넣기 가능 (Lv3 없이도) |

---

### Mountain (산) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 지휘관 **거인화** |
| 추천 | **보병** |
| 이유 | 업그레이드 시 브루트 1:1 가능, 탱킹 |

> ⚠️ **주의**: 업그레이드 전에는 효과 미미

---

### Ironskin (철피부) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 받는 데미지 감소 |
| 추천 | 범용 |

---

### Sure Footed (확고한 발) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 넉백/스턴/날림 저항 |
| 추천 | **보병** |
| 이유 | 보트 착륙에 안 넘어짐, 해변 배치 가능 |

**특수 효과**
- 보트 착륙 넉백 **면역**
- 브루트 넉백 저항
- 대형 활 넉백 저항

---

### Fearless (두려움 없음) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | **절대 도주하지 않음** |
| 평가 | ⚠️ **위험할 수 있음** |
| 이유 | 최선: 발동 안 함 / 최악: 유닛 전멸 |

---

## 유틸리티 특성 (4종)

### Energetic (정력적) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 모든 능력 쿨다운 **~33% 감소** |
| 시너지 | Warhammer (40초 → ~27초) |

---

### Fleet of Foot (빠른 발) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 분대 이동속도 **~33% 증가** |
| 활용 | 빠른 재배치, 다방면 방어 |

---

### Popular (인기) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 분대 최대 인원 **+1** |
| 활용 | Ring of Command와 중첩 |

---

### Rousing Speeches (울림 연설) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 집에서 보충 시간 **~33% 감소** |
| 활용 | 빠른 병력 복구 |

**수치**
- 병사당 1.34초 (기본 2초에서 감소)

---

## 경제 특성 (3종)

### Skillful (숙련) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 스킬 업그레이드 **50% 할인** |

**적용 비용**

| 스킬 레벨 | 기본 | Skillful 적용 |
|----------|------|---------------|
| Lv1 | 7 | **3.5 → 4** |
| Lv2 | 10 | **5** |
| Lv3 | 14 | **7** |

---

### Collector (수집가) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | 아이템 업그레이드 **50% 할인** |

---

### Heavy Load (무거운 짐) 🔵

| 항목 | 내용 |
|------|------|
| 효과 | Bomb/Mines/War Horn 사용 횟수 **+1** |
| 적용 | 레벨당 +1 |

---

## 특성-클래스 최적 조합

| 클래스 | 최적 특성 | 시너지 이유 |
|--------|----------|-------------|
| 보병 | Sure Footed | 해변 배치, 넉백 저항 |
| 보병 | Mountain | 탱킹, 브루트 대응 |
| 창병 | Sharp Weapons | 빠른 처치 = 방어선 유지 |
| 창병 | Energetic | Warhammer 쿨다운 감소 |
| 궁수 | Heavy Weapons | 물 밀어넣기 |
| 궁수 | Popular | 화살 수 증가 |

---

## 특성 효과 요약

| 특성 | 수치 |
|------|------|
| Energetic | 쿨다운 **-33%** |
| Fleet of Foot | 이동속도 **+33%** |
| Rousing Speeches | 보충시간 **-33%** |
| Skillful | 스킬 비용 **-50%** |
| Collector | 아이템 비용 **-50%** |
| Popular | 분대원 **+1** |

## 특성 해금 시스템

### 해금 방법
- 특정 섬 승리 시 해금

### 시각적 표시
- **잠금 특성**: 흰색 문장
- **해금 특성**: 배경색 있는 문장

### 시작 장비 시스템
- 해금된 특성/아이템
- 캠페인 시작 시 선택 가능
- 시작 지휘관 **2명**에게 부여
