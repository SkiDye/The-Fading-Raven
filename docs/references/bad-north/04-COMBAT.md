# Bad North - 전투 메카닉

## 핵심 메카닉

### 1. 창 들어올림 (Pike Vulnerability) ⚠️
- 창병이 **근접 격투 범위**에 들어가면 창을 들어올림
- 이 상태에서 **공격 불가**
- **해결책**: Warhammer로 밀어내기, 다른 유닛 엄호

### 2. 근접전 중 방패 무효 ⚠️
- 보병이 **근접 교전 중**일 때 방패로 화살 방어 불가
- 적 방패병도 동일 → **교전 중 궁수 사격 유효**

### 3. 물 즉사 ✅
- 물에 빠지면 **아군/적군 모두 즉사**
- 활용: Heavy Weapons 특성 궁수, Warhammer 넉백
- 주의: Brutes 넉백으로 아군 익사 위험

### 4. 보트 상륙 넉다운
- 보트 크기에 따라 넉다운 강도 변화
- **Sure-Footed 특성**: 넉다운 면역

## 보충 시스템 (Replenish) 🔵

### 기본 메카닉
1. 분대원 **1명 이상 손실** 시 사용 가능
2. **온전한 집** 선택
3. 시간 경과 후 풀 병력 복구

### 보충 시간 공식
```
보충 시간 = 2초 × 분대크기

기본 (9명): 2 × 9 = 18초
Ring Lv1 (12명): 24초
Ring Lv2 (16명): 32초
```

### 영향 요소
| 요소 | 영향 |
|------|------|
| 분대 크기 | 증가 → 시간↑ |
| Rousing Speeches | **-33%** (1.34초/병사) |
| 남은 병력 수 | **영향 없음** |
| 집 크기 | **영향 없음** |

### 보충 중 제한
- 명령 불가
- **취소 불가**
- 집 파괴 시 → **영구 손실** 가능

---

## 영구 사망 (Permadeath)

### 지휘관 무적 조건
- 병사 **1명 이상 생존** → 지휘관 무적
- 전 병사 사망해도 지휘관만 살면 다음 섬에서 복구

### 영구 손실 조건
| 상황 | 결과 |
|------|------|
| 보충 중 집 파괴 + 지휘관 분리 | **영구 손실** |
| 도주 실패 (보트 크기 부족) | **영구 손실** |
| 모든 지휘관 사망 | **게임 오버** |

---

## 도주 시스템 (Flee)

### 도주 방법
1. 빈 바이킹 보트 클릭
2. 분대 선택 후 보트로 이동

### 도주 조건
| 조건 | 설명 |
|------|------|
| 빈 보트 필요 | 적이 모두 내린 보트 |
| 보트 크기 ≥ 분대 크기 | 수용 가능해야 함 |
| 홈 아일랜드 불가 | 깃발 섬에서 도주 불가 |
| Fearless 불가 | 해당 특성 지휘관 도주 불가 |

### 특수 메카닉
- **궁수는 도주 중에도 사격 가능**
- 도주 명령 **취소 불가**

## 슬로우 모션 🔵

### 작동 방식
- **스페이스바 홀드**: 시간 감속
- **유닛 선택 시**: 자동 슬로우 모션

### 활용
- 전술 판단 시간 확보
- 정확한 타이밍 조절
- 복잡한 상황 정리

## 전투 전략 팁

### 초반 (1-10섬)
- 3-4명 Veteran 확보
- 쉬운 섬 우선

### 중반 (10-20섬)
- Elite 1-2명 확보
- 아이템 장착
- 지형 활용 숙달

### 후반 (20섬+)
- 풀 Elite 필수
- 궁수로 점퍼 대응
- 손실 최소화 운영
