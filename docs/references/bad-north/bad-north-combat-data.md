## 2. 유닛 특성

### 2.1 Militia (민병)
- 가장 약한 유닛
- 방패 없음 → 원거리 공격에 취약
- 업그레이드된 유닛보다 느림
- 메에 이니셔티브 낮음, 자주 넘어짐
- **강점**: Enemy Archers(근접), Enemy Recruits 상대 시 적절

### 2.2 Infantry (보병)
- 방패로 원거리 공격 차단 가능 (~90% 데미지 감소)
- **단, 근접 전투 중에는 원거리 차단 불가**
- Lv3: 근접에서 추가 피격 1회 버틸 수 있음
- Lv3: Berserkers와도 대등하게 싸울 수 있음
- **고유 스킬**: Plunge (점프 공격)
  - Lv1-2: 1-2칸 높이에서 점프
  - Lv3: 무제한 높이에서 점프, 범위 내 모든 적 즉사

### 2.3 Pikes (창병)
- 위치 방어에 특화
- **이동 중에는 전투 불가** ⚠️
- 내리막 방어 시 특히 강력
- **고유 스킬**: Pike Charge
  - Lv1: 3칸 거리 돌격
  - Lv2: 무제한 거리 돌격
  - Lv3: Brutes도 즉사

### 2.4 Archers (궁수)
- 유일한 원거리 유닛
- 근접에서 매우 취약
- 높은 지형에서 사거리 증가
- 이동 중 공격 가능 (개별 궁수는 발사 시 잠시 멈춤)
- **고유 스킬**: Volley (화살 비)
  - 방패 든 적도 처치 가능
- 업그레이드: 사거리, 발사 속도, 정확도 향상

## 3. 전투 메카닉

### 3.1 상성 관계 (Rock-Paper-Scissors)

```
┌─────────────┐
│   Archers   │ ←── Infantry (방패로 화살 차단)
└──────┬──────┘
       │ (무방비 적에게 효과적)
       ↓
┌─────────────┐
│   Militia   │ ←── 모든 유닛에 취약
└──────┬──────┘
       │
       ↓
┌─────────────┐
│    Pikes    │ ←── Archers/Huscarls (원거리 공격)
└─────────────┘
```

| 아군 | 강함 (vs) | 약함 (vs) |
|------|-----------|-----------|
| **Infantry** | Archers, 대부분 근접 | Brutes |
| **Pikes** | 모든 근접 (정지 시) | 원거리 공격, 이동 중 공격 |
| **Archers** | 방패 없는 적 | 근접 전투, 방패 든 적 |

### 3.2 방패 방어율
| 상황 | 방어율 | 출처 |
|------|--------|------|
| Infantry 방패 차단 | **~90%** 데미지 감소 | 🔵 Fandom |
| 근접 전투 중 | **0%** (차단 불가) | 🔵 Wikidot |

### 3.3 창병(Pikes) 무력화 조건
1. **이동 중 공격받음** → 전투 불가 상태
2. **원거리 공격** (화살, 도끼)
3. **측면/후방 공격** (방향 전환 전)
4. **대규모 적의 정면 돌진** (압도당함)

### 3.4 영구 사망 조건
- 🔵 **물에 빠지면 즉사** (익사)
  - 적 넉백으로 밀려남
  - 보트에서 떨어짐
- 🔵 **전투 중 HP 소진**
- 🔵 **건물이 불타는 중 Replenish 시도** → 사망

### 3.5 회복(Replenish) 메카닉
- 건물에 들어가면 유닛 회복
- Ring of Command 사용 시 회복 시간 증가
- 건물이 불타면 회복 불가
- **Berserkers는 Replenish 중인 유닛을 기다림** (공격 안 함)

## 4. 적 유닛

### 4.1 적 유닛 목록

| 이름 | 그룹 크기 | 특징 | 카운터 |
|------|-----------|------|--------|
| **Enemy Militia** (Recruits) | 3-18 | 방패 없음, 가장 약함 | Pikes, Archers |
| **Enemy Infantry** | 1-10 | 방패 있음 (~90% 방어) | Pikes, Lv2+ Infantry |
| **Enemy Archers** | 2-18 | 원거리 공격, 이동 중 사격 | Infantry |
| **Huscarls** (Normans) | 3-8 | 강화 방패, 투척 도끼 (1회) | Infantry |
| **Brutes** (Giants) | 2-8 | 고HP, 고데미지, 넉백, 방패 없음 | Pikes |
| **Dual Wielders** | 2-8+ | Berserkers 대체 (Jotunn Edition) | Pikes, 초크포인트 |
| **Brute Archers** | ? | 원거리 Brute 변형 | Infantry |

### 4.2 적 상세

#### Enemy Militia (Viking Recruits)
- 가장 약한 적
- 방패/원거리 공격 없음
- 대규모로 나타나 압도 시도
- 🔵 그룹: 3-18명, 드물게 1-2명

#### Enemy Infantry
- 방패로 화살 차단 (근접 중에는 불가)
- 🔵 그룹: 1-10명
- 카운터: Pikes (정지 상태)

#### Enemy Archers
- 보트 접근 중에도 사격
- 추격당하면 이동하며 사격
- **2칸 거리 유지 시도**
- 🔵 그룹: 2-18명

#### Huscarls (Normans)
- 강화된 Infantry + 투척 도끼
- 도끼: 1칸 높이 언덕까지 던질 수 있음
- **도끼 투척 중에는 원거리 차단 불가**
- 🔵 그룹: 3-8명
- 고유 BGM 있음

#### Brutes (Giants)
- **1타에 대부분 유닛 즉사**
- 높은 넉백 → 물에 빠뜨릴 수 있음
- 여러 번 피격 버팀
- 방패 없음 → 화살에 취약 (즉사는 아님)
- 🔵 그룹: 2-8명
- **건물보다 유닛 공격 우선**

#### Dual Wielders (Jotunn Edition)
- Berserkers 대체 유닛
- 쌍검 사용, 방패 없음
- 빠른 속도
- 공격적, 건물 무시하고 유닛 추격

#### (Legacy) Berserkers
- **Jotunn Edition에서 Dual Wielders로 대체됨**
- 강화된 Infantry, 방패 있음
- 더 빠르고, 더 높은 HP
- **Lv1 Plunge, Volley, Pike Charge에 저항**
- 🔵 그룹: 2-8명
- 건물 무시, 유닛 추격

## 5. 기타 메카닉

### 5.1 익사 (Drowning)
- 물에 빠지면 **즉사**
- 발생 조건:
  - 넉백으로 물에 밀려남
  - 보트에서 떨어짐 (화살 넉백 등)
  - Plunge/Pike Charge로 밀어냄

### 5.2 지형 효과
- **높은 지형**: Archers 사거리 증가
- **내리막**: Pikes 방어력 강화
- **초크포인트**: 모든 유닛 효율 증가

### 5.3 Jotunn Edition 변경사항
- Archers: 방패 없는 적/기절한 적 우선 타겟팅
- 보트 위 적도 밀어서 익사 가능 (버프)
- Berserkers → Dual Wielders 교체
