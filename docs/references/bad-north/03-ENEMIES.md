# Bad North - 적 유닛 시스템

## 적 분류 체계

```
기본 적 (Basic)
├── Viking Militia (검)
├── Viking Archers (활)
└── Viking Shieldmen (방패)

중급 적 (Advanced)
├── Huscarls (황색 방패 + 투척 도끼)
└── Dwarves (빠른 소형)

고급 적 (Elite)
├── Brutes (거인)
├── Brute Archers (거인 궁수)
└── Dual Wielders (쌍검)
```

---

## 기본 적

### Viking Militia 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **3-18명** |
| 체력 | 1타 사망 |
| 위협도 | ★☆☆☆☆ |
| 카운터 | 모든 유닛 |

### Viking Archers 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **2-18명** |
| 특징 | 이동 중 사격, 2타일 거리 유지 |
| 위협도 | ★★☆☆☆ |
| 카운터 | **Infantry** (방패) |

### Viking Shieldmen 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **1-10명** |
| 방어율 | **~90%** 데미지 감소 |
| 위협도 | ★★☆☆☆ |
| 카운터 | Pikes, 측면 공격, Volley |

> ⚠️ **근접 교전 중에는 방패 무효** → 궁수로 사격 가능

## 중급 적

### Huscarls (황색 방패) 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **3, 5, 8명** |
| 무기 | 검 + 방패 + **투척 도끼 1개** |
| 위협도 | ★★★★☆ |
| 카운터 | 협공, 특수 능력 |

**특수 메카닉**
- 도끼 투척: 1칸 높이까지 도달
- ⚠️ **투척 중 방패 내림** → 화살에 취약 (약 1초)

### Dwarves (난쟁이) 🟡
| 항목 | 값 |
|------|-----|
| 특징 | **매우 빠름**, 방패 소지 |
| 위협도 | ★★★☆☆ |
| 카운터 | 미리 배치 후 대기 |

## 고급 적

### Brutes (거인) 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **2-8명** |
| 체력 | 다수 타격 생존 |
| 특징 | **1타 즉사** 공격, 강한 넉백 |
| 위협도 | ★★★★★ |
| 카운터 | **Pikes** (최우선) |

**주의사항**
- 보병 정면 대응 금지
- 물가에서 넉백 → 아군 익사 위험
- Pikes Lv1로 **2-4마리 동시 방어** 가능

### Brute Archers 🔵
| 항목 | 값 |
|------|-----|
| 출현 | 단독 또는 2명 |
| 무기 | 발리스타급 거대 활 |
| 특징 | **1발 1-3명 관통** |
| 위협도 | ★★★★★ |
| 카운터 | Infantry 돌격 |

### Dual Wielders (점퍼) 🔵
| 항목 | 값 |
|------|-----|
| 그룹 크기 | **1-12명** |
| 무기 | 쌍검, 방패 없음 |
| 특징 | **점프 공격** (쿨다운 ~3초) |
| 위협도 | ★★★★★ |
| 카운터 | Archers (상륙 전), Pike Charge |

**특수 메카닉**
- Jotunn Edition에서 Ninja 대체
- 집보다 **유닛 공격 우선**
- 회복 중인 유닛 앞에서 대기

## 상성 관계 요약

### 기본 상성 (가위바위보)
```
Infantry(방패) ──────▶ 적 궁수/도끼병
      │
      │ 약함
      ▼
   Brutes ◀────────── Pikes(창)
                         │
                         │ 약함
                         ▼
                     적 궁수 ◀── Archers
                                    │
                                    │ 막힘
                                    ▼
                                적 방패병
```

### 상세 상성표
| 아군 | 강함 | 약함 |
|------|------|------|
| Infantry | 적 궁수, 도끼병 | Brutes |
| Pikes | Brutes, 일반 근접 | 적 궁수, 점퍼(근접) |
| Archers | 방패 없는 적 | 방패병, 빠른 적 |
