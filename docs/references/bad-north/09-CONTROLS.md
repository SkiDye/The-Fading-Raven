# Bad North - 조작법

## 개요

- 마우스와 키보드 또는 마우스만으로 플레이 가능
- 게임 내 별도의 조작법 설명 화면 없음 (커뮤니티 정보 기반)
- 키 리바인딩 불가

---

## 유닛 컨트롤

### 유닛 선택

| 조작 | 방법 |
|------|------|
| 마우스 클릭 | 유닛(분대) 직접 클릭으로 선택 |
| 숫자 키 **1-4** | 각 분대 빠른 선택 |

> ⚠️ 숫자 키 바인딩은 **고정**되어 있으며 재설정 불가

### 이동 명령

| 조작 | 방법 |
|------|------|
| 마우스 좌클릭 | 선택한 유닛을 목표 타일로 이동 |
| 드래그 | 유닛 선택 후 목표 위치로 드래그 |

---

## 카메라 컨트롤 🟡

### 카메라 회전

| 조작 | 방법 |
|------|------|
| 회전 버튼 **클릭** | 빠르게 90도 회전 (스냅 회전) |
| 회전 버튼 **홀드** | 천천히 부드럽게 회전 |
| **마우스 드래그** | 카메라 회전 (사용자 보고) |
| Q / E 키 | 카메라 좌/우 회전 (추정) |

> **참고**: 카메라는 드래그해서 회전한다는 사용자 피드백 있음 (커뮤니티 미확인)

### 줌 인/아웃

| 조작 | 방법 |
|------|------|
| 마우스 휠 위로 | 줌 인 |
| 마우스 휠 아래로 | 줌 아웃 |

### 카메라 특성
- **아이소메트릭 뷰**: 고정된 위에서 내려다보는 시점
- **회전 범위**: 360도
- **틸트/패닝**: 기본적으로 자동 (섬 전체가 화면에 맞춰짐)

---

## 게임 속도 컨트롤 🔵

### 슬로우 모션 / 일시정지

| 조작 | 방법 |
|------|------|
| **스페이스바 홀드** | 시간 감속 |
| 유닛 선택 시 | **자동으로 시간 느려짐** |

### 활용
- 전술 판단 시간 확보
- 정확한 타이밍 조절
- 복잡한 상황 정리

> FTL, Dungeon of the Endless처럼 필요할 때마다 시간을 멈추고 생각

---

## 스킬 사용

| 조작 | 방법 |
|------|------|
| 유닛 선택 후 스킬 버튼 클릭 | 스킬 활성화 |
| 목표 지점 클릭 | 스킬 발동 위치 지정 |

### 스킬 종류
- **Volley (궁수)**: 화살 집중 사격
- **Pike Charge (창병)**: 돌격
- **Plunge (보병)**: 다이브 공격

---

## UI 버튼

| 버튼 | 기능 |
|------|------|
| Deploy | 유닛 배치 |
| Skip Turn | 다음 웨이브로 건너뛰기 |
| Flee | 섬 탈출 (포기) |

> ⚠️ UI 버튼이 화면의 상당 부분을 차지하여 실수로 클릭할 수 있음 주의

| 조작 | 방법 |
|------|------|

| 출발 버튼 / Y키 | 선택한 섬으로 이동 |
