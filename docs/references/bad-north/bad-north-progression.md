# Bad North: Jotunn Edition - 로그라이트/진행 시스템 분석

## 개요
Bad North는 실시간 전술 로그라이트 게임으로, 바이킹 침략자로부터 섬을 방어하며 왕국을 탈출하는 게임입니다.

---

## 캠페인 구조

### 진행 방식
- **절차적 생성 맵**: 매 게임마다 새로운 섬 배치 생성
- **섬 연결 구조**: 오버월드 맵에서 섬 사이를 이동
- **바이킹 파도**: 턴이 끝날 때마다 바이킹 침략이 진행, 지나간 섬은 영구히 점령됨
- **다중 경로**: 플레이어가 다음 섬을 선택 가능 (FTL과 유사한 구조)
- **섬 개수**: 정확한 총 섬 개수는 비공개이나, 캠페인은 여러 "구역"을 거쳐 최종 섬까지 진행

### 섬 유형
- **일반 전투 섬**: 바이킹 방어
- **아이템 섬**: 물음표(?) 표시, 아이템 획득 가능
- **지휘관 섬**: 새로운 지휘관 영입 가능
- **체크포인트 섬** (Jotunn Edition): 진행 저장 가능

---

## 골드(코인) 시스템

### 획득 방식
- **집 방어 보상**: 각 섬의 집을 성공적으로 방어하면 코인 획득
- **피해에 따른 감소**: 집이 불타면 해당 집의 코인 감소 또는 상실
- **완전 파괴 시**: 해당 집에서 코인 0

### 코인 사용처
- **유닛 클래스 업그레이드**: 민병대 → 특화 병종 (방패병, 창병, 궁수)
- **스킬 해금**: 각 병종의 특수 능력
- **레벨 업**: 일반 → 베테랑 → 엘리트

### Jotunn Edition 변경사항
- **골드 공유**: 기존에는 각 지휘관별로 골드가 분리되었으나, Jotunn Edition에서 유닛 간 골드 공유 가능

---

## 유닛 시스템

### 기본 병종 (3종)
| 병종 | 강점 | 약점 |
|------|------|------|
| **방패병 (Infantry)** | 방어력 우수, 화살 방어 가능 | 공격력 낮음 |
| **창병 (Pikemen)** | 고정 방어 최강, 적 밀어내기 | 이동 중 공격 불가, 화살에 취약 |
| **궁수 (Archers)** | 원거리 공격, 다수 적 제압 | 근접전 취약 |

### 특수 스킬 (각 병종 1개씩)
- **방패병**: 절벽 돌진 (Plunge) - 높은 곳에서 뛰어내려 적 공격
- **창병**: 돌격 (Charge) - 직선으로 돌진 공격
- **궁수**: 화살 비 (Volley) - 집중 사격

### 유닛 등급
1. Militia (민병대) - 기본
2. Veteran (베테랑)
3. Elite (엘리트)

## 아이템 시스템

### 아이템 종류 (확인된 목록)
게임 내 아이템은 지휘관에게 장착하여 추가 능력을 부여합니다.

| 아이템 | 효과 |
|--------|------|
| **War Horn** | 아군 사기 증가/버프 |
| **Ring of Command** | 유닛 크기 2배 증가 |
| **Bomb/Mine** | 설치형 폭발물 |
| **Hammer** | 추가 공격 능력 |
| **Speed Boots** | 이동 속도 증가 |

*참고: 전체 아이템 목록은 공식 위키 접근 불가로 불완전함*

### 트레잇 시스템 (Jotunn Edition 추가)
- 지휘관에게 부여되는 소형 패시브 버프
- 예시: Tough (더 강함), Fast (더 빠름), Fearless (퇴각 불가)
- 일부 트레잇과 아이템은 영구 언락 가능

## Jotunn Edition 체크포인트 시스템

### 작동 방식
- **체크포인트 섬**: 특정 섬에서 진행 상황 저장 가능
- **사망 시 복귀**: 모든 지휘관을 잃으면 마지막 체크포인트로 복귀
- **기존과의 차이**: 원본 게임은 사망 시 완전히 처음부터 시작

### 중요사항
- 체크포인트는 런 중 특정 지점에서만 생성됨
- 체크포인트 사이에서 죽으면 해당 체크포인트로 롤백

---

## 영구 언락 요소 (Meta-Progression)

### Jotunn Edition에서 추가된 메타 진행
- **시작 업그레이드**: 이전 런에서 해금한 트레잇/아이템을 새 런 시작 시 장착 가능
- **두 명의 시작 지휘관**: 게임 시작 시 선택한 두 지휘관에게 해금된 아이템/트레잇 적용
- **언락 조건**: 특정 아이템과 트레잇은 여러 캠페인을 통해 점진적으로 해금

### 언락 가능 요소
- 일부 트레잇
- 일부 아이템
- Very Hard 난이도 (Hard 클리어 시)

---

## 게임 오버 조건

### 즉시 게임 오버
- **모든 지휘관 사망**: 지휘관이 한 명도 남지 않으면 게임 종료
- **체크포인트 없이 전멸**: Jotunn Edition 이전 버전 또는 체크포인트 미도달 상태

### 지휘관 영구 사망
- 전투 중 유닛이 전멸하면 해당 지휘관 **영구 사망 (Permadeath)**
- 사망한 지휘관은 해당 런에서 복구 불가

### 철수 시스템
- 전투 중 배를 타고 철수 가능
- 철수 시 해당 섬의 코인 획득 불가
- 생존을 위한 전략적 선택
