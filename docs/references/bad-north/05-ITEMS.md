# Bad North - 아이템 시스템

## 개요

- 지휘관에게 장착하여 추가 능력 부여
- 아이템 섬(**❓ 물음표** 표시)에서 획득
- 일부 아이템은 업그레이드 가능

---

## 아이템 목록 (10종)

### Ring of Command (지휘의 반지) ⭐

| 항목 | 내용 |
|------|------|
| 유형 | 패시브 |
| 효과 | 분대 최대 인원 증가 |
| 추천 유닛 | **궁수** |

**업그레이드**

| 레벨 | 분대 크기 | 보충 시간 | 비용 |
|------|----------|----------|------|
| Lv1 | 9명 → **12명** | 18초 → 24초 | 기본 |
| Lv2 | 12명 → **16명** | 24초 → 32초 | **16 골드** |

> 궁수 화살 수 = 분대원 수 → 화력 직결

**주의사항**
- 보충 시간 증가
- 여러 웨이브 대응 어려울 수 있음

---

### Warhammer (전쟁 망치) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 액티브 |
| 효과 | 전방 점프 → AOE 데미지 + 넉백 |
| 쿨다운 | **40초** |
| 추천 유닛 | **창병** |

**활용**
- 방어선 돌파한 적 밀어내기
- 보트 상륙 직후 적 물에 밀어넣기
- 창병 재정렬 시간 확보

**시너지**
- Energetic 특성: 쿨다운 ~27초 → "토르" 빌드

---

### Bomb (폭탄) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 액티브 |
| 효과 | 투척형 폭발물 |
| 사거리 | 1~2 타일 |
| **아군 피해** | **있음 (Friendly Fire)** |
| 추천 유닛 | **궁수** |

**업그레이드**

| 레벨 | 사용 횟수 | 비용 |
|------|----------|------|
| Lv1 | 1개/섬 | 기본 |
| Lv2 | 2개/섬 | **8 골드** |
| Lv3 | 3개/섬 + AOE↑ | **14 골드** |

**활용**
- 방패병 카운터
- 군중 제어
- 보트 상륙 지점

**주의사항**
- 대각선 투척 시 벽/집에 막힘
- 아군 피해 주의

---

### Mines (지뢰) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 액티브 |
| 효과 | 설치형 함정 → 적 접근 시 폭발 |

**업그레이드**

| 레벨 | 사용 횟수 | 비용 |
|------|----------|------|
| Lv1 | 1개/섬 | 기본 |
| Lv2 | 2개/섬 | **8 골드** |
| Lv3 | 3개/섬 + AOE/데미지↑ | **14 골드** |

**활용**
- 병목 지점 설치
- 적 필수 경로에 배치

**주의사항**
- ⚠️ **보트 착륙 충격으로 오작동 가능** (적에게 피해 없이 폭발)
- 타이밍 중요

### War Horn (전쟁 나팔) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 액티브 |
| 효과 | 즉시 병력 보충 |
| 추천 유닛 | **보병** |

**특징**
- 집 점거 없이 현장에서 보충
- 전투 중 손실 즉시 복구

**비교**
- Ring of Command 대비: 풀업 시 순 +3명 효과

---

### Holy Grail (성배) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 패시브 |
| 효과 | 사망한 유닛 **1회 부활** |
| 제한 | **게임당 1회만 사용** |

**제한사항**
- 섬 포기로 인한 도주 중 사망은 부활 불가

**활용**
- 핵심 Elite 지휘관 보험
- 최후의 수단

### Philosopher's Stone (현자의 돌) 🔵

| 항목 | 내용 |
|------|------|
| 유형 | 패시브 |
| 효과 | 섬 클리어 시 추가 골드 획득 |
| 조건 | **해당 유닛 배치 필수** |

**업그레이드**

| 레벨 | 추가 골드 | 비용 |
|------|----------|------|
| Lv1 | +1 골드/섬 | 기본 |
| Lv2 | +2 골드/섬 | **5 골드** |
| Lv3 | +3 골드/섬 | **9 골드** |

**전략**
- 초반 투자 시 수익 극대화
- 단, 전투력 부족하면 의미 없음
- 밸런스 있게 투자
