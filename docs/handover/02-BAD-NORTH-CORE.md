# Bad North 핵심 메카닉

> **중요**: Bad North가 이 게임의 레퍼런스. 핵심 메카닉은 반드시 유지.

---

## 필수 유지 메카닉

### 1. 가위바위보 상성
```
보병(실드) → 적 궁수/도끼병
    ↓ (약함)
브루트 ← 창병(랜스)
            ↓ (약함)
         적 궁수 ← 아군 궁수 ─X─ 적 방패병
```

### 2. 영구 사망
- 팀장(지휘관) 사망 시 분대 영구 손실
- 병사가 1명이라도 생존하면 팀장은 무적

### 3. 슬로우 모션
- 크루 선택 시 게임 속도 감소
- 전술 판단 시간 확보

### 4. 핵심 전투 메카닉
| 메카닉 | 설명 |
|--------|------|
| **창 들어올림** | 적이 밀착하면 창병(센티넬) 무력화 |
| **교전 중 실드 무효** | 근접전 중 방패로 화살 방어 불가 |
| **물 즉사** | 우주 공간 = 즉사 지역 |
| **보충 시스템** | 시설 점거 → 시간 경과 → 병력 복구 |

### 5. 웨이브 기반 방어
- 침투선(보트) 도착 → 적 상륙 → 격퇴
- 시설 방어 성공 = 자원 획득

### 6. 절차적 생성
- 시드 기반 RNG
- 동일 시드 = 동일 맵
