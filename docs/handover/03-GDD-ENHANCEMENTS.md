# GDD 개선사항

> **중요**: Bad North 기반이지만 GDD도 충분히 개선해서 작성됨. 둘 다 동등하게 중요.

---

## 신규 클래스 (2종)

### 엔지니어 (Engineer)
| 항목 | 내용 |
|------|------|
| 역할 | 지원, 터렛 설치, 시설 수리 |
| 분대 크기 | 6명 (타 클래스보다 적음) |
| 스킬 | Deploy Turret |
| 특수 | 파괴된 시설 수리 가능 (20초) |
| 주의 | 적 해커가 터렛 해킹 가능 |

### 바이오닉 (Bionic)
| 항목 | 내용 |
|------|------|
| 역할 | 고기동 암살 |
| 분대 크기 | 5명 |
| 스킬 | Blink (순간이동) |
| 특수 | 비교전 적 공격 시 2배 데미지 |
| 용도 | 스나이퍼, 드론 캐리어 등 우선순위 타겟 제거 |

---

## 신규 적 유닛

| 적 | 특징 | 카운터 |
|----|------|--------|
| **해커** | 터렛/시스템 해킹 | 우선 처치 필수 |
| **폭풍 생명체** | 자폭, 폭풍 스테이지 전용 | 원거리 처치 |
| **스나이퍼** | 맵 전역 공격, 3초 조준 후 즉사 | 바이오닉 암살 |
| **드론 캐리어** | 10초마다 드론 2기 소환 | 본체 우선 처치 |
| **실드 제너레이터** | 주변 적에게 실드 부여 | 최우선 처치 |
| **해적 대장** (보스) | 15히트, 주변 적 버프 | 집중 공격 |
| **폭풍 핵** (보스) | 파괴 불가, 환경 위험 | 웨이브 클리어로 퇴각 |

---

## 다양한 시설 모듈

| 시설 | 크레딧 | 특수 효과 |
|------|--------|----------|
| 거주 모듈 | 1~3 | 기본 |
| 의료 모듈 | 2 | 회복 시간 -50% |
| 무기고 | 2 | 데미지 +20% |
| 통신탑 | 1 | Raven 드론 능력 +1회 |
| 발전소 | 3 | 터렛 성능 +50% |

---

## Raven 드론 시스템 (핵심 차별점)

| 능력 | 효과 | 기본 횟수 |
|------|------|----------|
| 정찰 (Scout) | 다음 웨이브 구성 미리 보기 | 매 웨이브 |
| 조명탄 (Flare) | 폭풍 스테이지 시야 확보 | 2회 |
| 긴급 보급 (Resupply) | 1팀 즉시 회복 | 1회 |
| 궤도 폭격 (Orbital Strike) | 지정 타일 고데미지 | 1회 |
