## TFR 핵심 차별점 (⭐ 최우선 구현)

### Raven 드론 시스템

| 능력 | 효과 | 기본 횟수 |
|------|------|----------|
| **Scout (정찰)** | 다음 웨이브 구성 미리 보기 | 매 웨이브 |
| **Flare (조명탄)** | 폭풍 스테이지 시야 확보 (10초) | 2회 |
| **Resupply (긴급 보급)** | 1팀 즉시 회복 | 1회 |
| **Orbital Strike (궤도 폭격)** | 지정 타일 고데미지 | 1회 |

### 시설 보너스 시스템

| 시설 | 크레딧 | 보너스 효과 |
|------|--------|------------|
| 거주 모듈 | 1~3 | 없음 (기본) |
| **의료 모듈** | 2 | Resupply 시간 **-50%** |
| **무기고** | 2 | 데미지 **+20%** |
| **통신탑** | 1 | Raven 능력 **+1회** |
| **발전소** | 3 | 터렛 성능 **+50%** |

## TFR 재설계 시스템

### 긴급 귀환 (Emergency Evac)
> 기존 Flee → Raven 기함 셔틀 회수로 변경

```
[Emergency Evac 버튼]
       ↓
[Raven 셔틀 하강 (5초)]
       ↓
[분대 탑승 → 이륙]
       ↓
[크레딧 0, 크루 생존]
```

### 구조 임무 (RESCUE)
> 기존 깃발 섬 → 탈출자 보호 전투로 변경

```
[RESCUE 노드 선택]
       ↓
[탈출자가 시설에서 대기]
       ↓
[전투 중 탈출자 보호]
       ↓
[승리 → 새 팀장 + 크루 4명 합류]
```

### 노드 타입

| 노드 | 설명 |
|------|------|
| STATION | 기본 전투 |
| **RESCUE** | 구조 임무 → 팀장 영입 |
| **SALVAGE** | 난파선 탐색 → 장비 획득 |
| **DEPOT** | 보급 정거장 → 무료 장비 |
| **STORM_STATION** | 폭풍 스테이지 (시야 제한) |
| **BEACON** | 비콘 활성화 (체크포인트) |
| SHOP | 상점 |
| REST | 휴식 + 회복 |
| BOSS | 보스 전투 |
| GATE | 최종 점프 게이트 |
